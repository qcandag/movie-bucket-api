# BUCKET-MOVIE API

In this project I made an API with Adonis.Js and I used TDD technique.

## Installation

You should already have node and npm.

```bash
npm install
```

To start HTTP server:

```bash
node ace serve
```

To start tests:

```bash
node ace test
```
